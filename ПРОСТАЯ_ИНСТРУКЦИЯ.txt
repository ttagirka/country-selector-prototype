📌 ШАГ 1: Создать репозиторий на GitHub
─────────────────────────────────────────────────────────────
1. Откройте https://github.com
2. Нажмите "+" → "New repository"
3. Имя: country-selector-prototype
4. Нажмите "Create repository"

📌 ШАГ 2: Загрузить файлы на GitHub
─────────────────────────────────────────────────────────────
1. На странице репозитория нажмите "uploading an existing file"
2. Откройте Finder: /Users/ttikeev/Desktop/test/5_prototype/country-selector-prototype
3. Перетащите ВСЕ файлы в браузер (на страницу GitHub)
4. Нажмите "Commit changes"

📌 ШАГ 3: Подключить к Vercel
─────────────────────────────────────────────────────────────
1. Откройте https://vercel.com/new
2. Вставьте URL репозитория (скопируйте с GitHub)
   Или найдите репозиторий в списке "Import Git Repository"
3. Нажмите "Deploy"
4. Готово! Скопируйте URL из Vercel
